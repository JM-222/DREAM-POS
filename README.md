# 드림편의점 POS 시스템 🏪

**대동제 2026 드림편의점 POS 시스템** - 아이패드 및 모바일 최적화 웹 기반 판매 관리 시스템

---

## 📋 개요

이 프로젝트는 편의점 운영을 위한 완벽한 POS(Point of Sale) 시스템입니다. 아이패드와 노트북에서 모두 완벽하게 작동하며, 카카오톡으로 공유하여 즉시 사용할 수 있습니다.

**주요 기능:**
- 🛒 상품 관리 및 카트 시스템
- 💰 현금/카드 결제 처리
- 🎮 게임 목표 추적 (판매량 관리)
- 📊 실시간 매출 집계
- 🔊 결제 음향 효과
- 📱 아이패드/모바일 최적화
- 🌐 인터넷 연결 없이 오프라인 작동

## 🚀 사용 방법

### 1️⃣ 로컬에서 실행하기

**가장 간단한 방법 - 파일 열기:**
```bash
# index.html을 웹 브라우저에서 열기
# (Chrome, Safari, Firefox 등 모든 브라우저 지원)
```

**로컬 웹 서버로 실행 (권장):**
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# 그 후 브라우저에서 http://localhost:8000 접속
```

### 2️⃣ 아이패드에서 사용하기

**방법 1: 카카오톡으로 공유 (가장 편함)**
1. `index.html` 파일을 카카오톡에서 본인에게 전송
2. 아이패드 카카오톡에서 파일 클릭
3. Safari에서 열기 선택
4. 공유 버튼 → "홈 화면에 추가" 클릭
5. 홈 화면에 생성된 아이콘으로 앱처럼 사용

**방법 2: 웹 서버에 배포**
1. 이 폴더 전체를 웹 호스팅 서비스에 업로드 (GitHub Pages, Netlify 등)
2. 생성된 URL을 카카오톡으로 공유
3. 아이패드 Safari에서 링크 열기
4. "홈 화면에 추가" 클릭

### 3️⃣ 노트북/데스크톱에서 사용하기

```bash
# 1. 이 폴더를 웹 서버로 실행
python -m http.server 8000

# 2. 브라우저에서 접속
# http://localhost:8000
```

## 🎮 기본 사용법

### 상품 추가
1. 오른쪽 "상품" 탭에서 원하는 상품 클릭
2. 수량 조절 (+ / -)
3. 자동으로 왼쪽 카트에 추가됨

### 결제 처리
1. 왼쪽 카트에서 상품 확인
2. 하단 "현금" 또는 "카드" 버튼 클릭
3. 결제 완료 화면 표시
4. 화면 클릭하여 다음 주문 진행

### 게임 목표 추적
- 상단 "게임 바"에서 판매량 목표 추적
- 목표 달성 시 자동 표시

## 🔊 음향 효과

시스템에 포함된 음향 효과:
- **Beep**: 버튼 클릭음
- **Cash**: 현금 결제 완료음
- **Card**: 카드 결제 완료음
- **Success**: 성공 알림음
- **Fail**: 실패 알림음

음향 효과는 사용자의 첫 터치 후 자동으로 활성화됩니다.

## 🐛 문제 해결

### 음향이 안 나올 때
- 기기의 음량을 확인하세요
- 사파리 설정에서 음향 허용 확인
- 화면을 한 번 터치한 후 다시 시도

### 이미지가 안 보일 때
- 폴더 구조가 올바른지 확인
- 브라우저 캐시 삭제 후 새로고침
- 개발자 도구(F12)에서 콘솔 오류 확인

### 아이패드에서 느릴 때
- 다른 앱 종료
- 브라우저 캐시 삭제
- 기기 재부팅
